📦 Install (30 seconds)

  1. Download the ZIP from GitHub
  2. Unzip to any folder
  3. Chrome → chrome://extensions → Enable "Developer mode" (top right)
  4. Click "Load unpacked" → select the unzipped folder
  5. Done! Press Ctrl+Shift+P to try it
